# Notes

## Serving the website locally

Install Jekyll:
```{sh}
sudo pacman -S ruby-jekyll
```
Then run:
```{sh}
jekyll serve
```

Alternatively, Jekyll can be installed in a user-local
[Ruby Gem environment](https://wiki.archlinux.org/title/Ruby#RubyGems)
and run with [bundler](https://wiki.archlinux.org/title/Ruby#Bundler).
